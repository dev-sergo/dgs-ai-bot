## Чек-лист плана работ (зафиксирован 2026-06-25)

### Трек A — Q2 «прогноз выручки» (ПЕРВЫМ; на live payment, нового клиента НЕ требует)
- [ ] **A1.** Движковая функция прогноза в `internal/engine`: run-rate = темп закрытой части периода ×
      остаток периода, поверх payment-ряда по дням. Детерминированно, источник чисел — payment (live/фикстура).
- [ ] **A2.** Честная оговорка о точности: мало данных / самое начало периода / неполные дни → явный
      дисклеймер; не выдавать точную цифру как факт.
- [ ] **A3.** Граничные случаи: пустой период, 1 день данных, уже закрытый период (факт, не прогноз),
      будущий/целиком незакрытый период.
- [ ] **A4.** Цель/план: если в реплике названа цель (X ₽) — сравнить прогноз с ней («дойдёшь/не дойдёшь,
      разрыв N»); если нет — просто проекция. Решить: спрашивать план или нет.
- [ ] **A5.** Роутинг: «дойду ли до плана», «какая будет выручка к концу месяца», «если ничего не менять» →
      forecast. **Узкий детерминированный guard** (по образцу `planner.RefineAdvice`), НЕ фаззи-правило в промпте.
- [ ] **A6.** Рендер/нарратив: текст с прогнозом + оговоркой; число — из движка (sandwich).
- [ ] **A7.** Тесты: engine-unit с предрасчётом на фикстуре + pipeval-кейсы; Тир 0 зелёный (`make vet && make test`).
      Если правится промпт планировщика → полный `eval-host` (пользователь) до коммита.

## Рабочие правила (важно соблюдать)
- **Риг-прогоны запускаю Я (пользователь), не ты.** Ты готовишь Тир 0 (юнит/`make test`/`vet`) и отдаёшь
  команду; реальный LLM (`make eval-host`/`pipeval-host`) гоняю я и присылаю вывод.
- **Любая правка промпта планировщика → полный `eval-host` до коммита** (узкий фикс уже ронял 523→507).
  Чисто движковые правки — только `make test`. Go запускается через Docker (`make test`), бинаря `go` нет.
- **Прогоны сохранять в файл:** `… 2>&1 | tee <name>-$(date +%Y%m%d-%H%M).log`.
- **Git-коммиты на английском, без Co-Authored-By.** Коммитить/пушить — только когда прошу;
  коммитить ТОЛЬКО файлы своего трека, не задевая параллельную работу (Telegram).
- **Стенд:** деплой `docker compose up --build` запускаю я; бэкенд `bot` на :8088, веб на :8090,
  снаружи — bot.bubnov.site. Токен демо-гейта — у меня.
- **Не приступай к коду, пока не согласуем шаг внутри трека.**
